# Unsplash API Project

This project is a simple web application that utilizes the Unsplash API to display images based on user queries.

## Credits

This project relies on the [Unsplash API](https://unsplash.com/developers) to fetch high-quality images. Unsplash provides a vast collection of freely-usable images contributed by photographers worldwide.

## Features

- Users can enter a search query in the provided input field.
- Upon submitting the form, the application fetches images related to the user's query from the Unsplash API.
- The retrieved images are displayed on the webpage in a responsive grid layout.

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Clone this repository to your local machine.
2. Open the `index.html` file in your web browser.
3. Enter a search query in the input field and submit the form.
4. View the fetched images displayed on the webpage.
